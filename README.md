# DunDuns for DumDums

![Interface](https://img.shields.io/badge/Interface-12.0.0-blue)
![Game](https://img.shields.io/badge/Game-WoW_Retail-orange)
![Patch](https://img.shields.io/badge/Expansion-Midnight-purple)

A lightweight World of Warcraft addon for **Midnight** that tracks your **Shard of Dundun** and **Unalloyed Abundance** currencies while providing one-click waypoints to big abundance events.

![Screenshot](https://your-image-host.com/screenshot.png) *(Add your screenshot here)*

## Features

- **Live Currency Tracking**: Real-time counters for Shard of Dundun (center icon) and Unalloyed Abundance (bottom-left)
- **One-Click Navigation**: Instantly drop map pins for Midnight's four big abundance event locations:
  - Eversong
  - Zul'Aman  
  - Harandar
  - Voidstorm
- **Clear Pins**: Bottom-right button to remove waypoints when done
- **Smart UI**: Automatically hides when opening the world map to prevent clutter
- **Persistent Positioning**: Drag anywhere; saves location between sessions
- **Zero Config**: Works out of the box with no setup required

## Installation

### CurseForge (Recommended)
Search for "DunDuns for DumDums" and click Install.

### Manual
1. Download the latest release
2. Extract to `World of Warcraft\_retail_\Interface\AddOns\`
3. Rename folder to `DunDunsForDumDums` (if needed)
4. Restart WoW or type `/reload`

## Usage

### Slash Commands
- `/dunduns` or `/dd` - Toggle the addon window

### Buttons
- **Eversong / Zul'Aman / Harandar / Voidstorm**: Sets a waypoint to the big abundance event in that zone
- **Clear Pins**: Removes the current waypoint and stops tracking it
- **X**: Close the window (use `/dd` to reopen)
